## рҹҺҜ Fitur Lengkap

### 1пёҸвғЈ рҹ“Ў Monitor вҖ” Pantau Domain Anti-Nawala

- вң… Scan ke **TrustPositif Kominfo** (sumber resmi)
- вң… Multi-source check: **Kominfo + TrustPositif API + NawalaCheck** (triple-check)
- вң… **Auto rotating batch** вҖ” handle 100+ domain tanpa rate-limit
- вң… Sticky cache вҖ” domain blocked gak di-check ulang (hemat API)
- вң… Force-block manual untuk testing
- вң… Spam alert continuous sampai user hapus dari list

### 2пёҸвғЈ вҡҷпёҸ CF Redirect вҖ” Cloudflare Auto-Manage

- вң… Auto-discovery rule pakai nama domain (no Zone ID manual)
- вң… Support **V1 Page Rules** & **V2 Redirect Rules**
- вң… Add domain baru ke Cloudflare lengkap dengan DNS placeholder
- вң… Bulk change URL (banyak rule sekaligus)
- вң… Preserve path & query string saat swap

### 3пёҸвғЈ рҹ”„ Auto Rotator вҖ” Unified Hub (CF + Klikcepat)

- вң… Setup Rotator вҖ” pilih tipe (CF / Klikcepat) вҶ’ pick rule вҶ’ pool
- вң… **Bulk Setup** вҖ” checkbox picker, assign banyak rule ke 1 pool
- вң… List Rotator grouped view (CF + Klikcepat dengan totals)
- вң… Swap history log
- вң… Reactive auto-swap < 1 menit dari blocked detection

### 4пёҸвғЈ рҹ”— KLIKCEPAT вҖ” Bio Link & Short URL CRUD

- вң… Full CRUD link (biolink + shortlink + vcard + event)
- вң… Project management (grouping link)
- вң… Filter view by type (Biolink only / Shortlink only)
- вң… Auto-detect custom domain (klikcepat.com, klikcepat.vip, custom)
- вң… Auto-swap location_url ketika target blocked
- вң… Integrasi dengan Auto Rotator unified

### 5пёҸвғЈ рҹ”§ Settings вҖ” Hub Picker

- вң… Pilih platform (CF / Klikcepat)
- вң… Set credentials via bot (no edit .env)
- вң… Test koneksi per platform
- вң… Auto-default klikcepat.com
- вң… Credentials encrypted di `data/credentials.json`

## рҹҗӣ Troubleshooting

<details>
<summary><b>Bot gak respond ke /start</b></summary>

```bash
sudo journalctl -u bongbot -n 50 --no-pager
```

**Cek error spesifik:**
- `BOT_TOKEN tidak di-set` вҶ’ cek `.env`
- `Bot error: Unauthorized` вҶ’ token salah / di-revoke di BotFather
- `permission denied` вҶ’ `sudo chown -R bongbot:bongbot /opt/bongbot`
</details>

<details>
<summary><b>Klikcepat "Credentials belum lengkap"</b></summary>

Set API Key via `рҹ”§ Settings вҶ’ рҹ”— Klikcepat вҶ’ рҹ”‘ Set API Key`.
Base URL auto-default ke `https://klikcepat.com`, gak perlu set manual.
</details>

<details>
<summary><b>Setup Rotator gak berfungsi</b></summary>

Deploy binary terbaru:
```bash
sudo bash /opt/bongbot/deploy/install.sh
```
Mungkin masih pakai binary lama dari sebelum FlexInt fix.
</details>

<details>
<summary><b>Markdown parse error di Telegram</b></summary>

Underscore (`_`) di text dynamic ke-interpret sebagai italic.
Reproduce: kalau pesan dynamic punya kata dengan `_`, escape via backticks.
</details>

<details>
<summary><b>Zombie bot process setelah reset</b></summary>

```bash
sudo pkill -9 -f bot-linux-amd64
sudo systemctl restart bongbot
```
Penyebab: `rm -rf /opt/bongbot` tanpa stop service dulu вҶ’ process keep running pakai deleted binary.
</details>
